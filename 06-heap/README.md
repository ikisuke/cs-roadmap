# ヒープ (Heap)

## 概要
完全二分木の性質を持つ優先度付きキュー

## 種類
- **最大ヒープ**: 親 ≥ 子（根が最大値）
- **最小ヒープ**: 親 ≤ 子（根が最小値）

## 時間計算量
- 挿入: O(log n)
- 削除（最大/最小値）: O(log n)
- 最大/最小値の取得: O(1)
- ヒープ構築: O(n)

## 配列実装での親子関係
- 親のインデックス: (i-1)/2
- 左の子のインデックス: 2*i+1
- 右の子のインデックス: 2*i+2
