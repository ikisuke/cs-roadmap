# 🚀 コンピューターサイエンス学習ロードマップ

## 📖 概要

週10時間の学習で、18-24ヶ月かけてコンピューターサイエンスの基礎を体系的に身につけるための包括的なロードマップです。理論学習と実践プロジェクトを組み合わせた効果的なカリキュラムを提供します。

## 🎯 [**インタラクティブ・ビジュアルロードマップ**](https://ikisuke.github.io/cs-roadmap/)

**[📈 index.html](index.html)** - クリックして学習進捗を追跡できるビジュアルロードマップ

### 主な機能
- **📊 進捗追跡**: 各フェーズをクリックして学習状況を記録
- **🎨 ビジュアル表示**: タイムラインとプログレスバーで学習計画を視覚化
- **📱 レスポンシブ**: PC・タブレット・スマホに対応
- **💾 自動保存**: ブラウザに学習進捗を自動保存

## 📚 6つの学習フェーズ

### Phase 1: プログラミング基礎と数学 (1-4ヶ月)
- **プログラミング基礎**: 変数、制御構造、関数、OOP
- **離散数学**: 論理、集合、グラフ理論
- **推奨リソース**: Harvard CS50、paiza

### Phase 2: データ構造とアルゴリズム (5-8ヶ月)
- **基本データ構造**: 配列、リスト、スタック、キュー
- **高度な構造**: 木、グラフ、ハッシュテーブル  
- **アルゴリズム**: ソート、探索、動的計画法
- **推奨リソース**: AtCoder、LeetCode

### Phase 3: コンピュータアーキテクチャ (9-11ヶ月)
- **デジタル論理**: 論理ゲート、数値表現
- **プロセッサ**: 命令セット、パイプライン
- **メモリ階層**: キャッシュ、仮想メモリ
- **推奨リソース**: RISC-V、nand2tetris

### Phase 4: オペレーティングシステム (12-14ヶ月)
- **プロセス管理**: スケジューリング、スレッド
- **メモリ管理**: 仮想メモリ、ページング
- **ファイルシステム**: ストレージ、アクセス制御
- **推奨リソース**: OSTEP、xv6 OS

### Phase 5: ネットワークとデータベース (15-18ヶ月)
- **ネットワーク基礎**: TCP/IP、HTTP、DNS
- **分散システム**: CAP定理、レプリケーション
- **データベース**: SQL、NoSQL、ACID
- **推奨リソース**: Wireshark、MongoDB

### Phase 6: ソフトウェアエンジニアリング (19-24ヶ月)
- **開発手法**: アジャイル、TDD、CI/CD
- **設計パターン**: SOLID、アーキテクチャ
- **クラウド技術**: AWS、Docker、Kubernetes
- **AI/ML統合**: 機械学習基礎、AIツール活用

## ⚡ 時間計算量早見表

| データ構造 | 検索 | 挿入 | 削除 |
|------------|------|------|------|
| 配列 | O(n) | O(n) | O(n) |
| 連結リスト | O(n) | O(1) | O(1) |
| スタック | - | O(1) | O(1) |
| キュー | - | O(1) | O(1) |
| 二分探索木 | O(log n) | O(log n) | O(log n) |
| ハッシュテーブル | O(1) | O(1) | O(1) |

| ソートアルゴリズム | 平均 | 最悪 | 空間 |
|-------------------|------|------|------|
| バブルソート | O(n²) | O(n²) | O(1) |
| マージソート | O(n log n) | O(n log n) | O(n) |
| クイックソート | O(n log n) | O(n²) | O(log n) |
| ヒープソート | O(n log n) | O(n log n) | O(1) |

## 🎯 学習の進め方

### 週10時間の効果的な配分
- **平日**: 1.5時間×5日 = 7.5時間（理論学習+コーディング）
- **週末**: 2.5時間（プロジェクト実装）

### フェーズ別時間配分
**Phase 1-2（基礎構築期）**
- 動画講義視聴：4時間
- コーディング練習：4時間  
- 読書・理論学習：2時間

**Phase 3-5（専門深化期）**
- 理論学習：3時間
- ハンズオン実習：5時間
- プロジェクト開発：2時間

**Phase 6（統合期）**
- 最新技術調査：2時間
- 実践プロジェクト：6時間
- コミュニティ参加：2時間

## 💻 コンパイルと実行

```bash
# 基本的なコンパイル
gcc filename.c -o output

# 数学ライブラリを使用する場合
gcc filename.c -o output -lm

# 実行
./output
```
