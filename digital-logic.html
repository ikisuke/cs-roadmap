📐 ブール代数の基礎

基本法則

結合法則

A ∧ (B ∧ C) = (A ∧ B) ∧ C
A ∨ (B ∨ C) = (A ∨ B) ∨ C

交換法則

A ∧ B = B ∧ A
A ∨ B = B ∨ A

分配法則

A ∧ (B ∨ C) = (A ∧ B) ∨ (A ∧ C)
A ∨ (B ∧ C) = (A ∨ B) ∧ (A ∨ C)

ド・モルガンの法則

¬(A ∧ B) = ¬A ∨ ¬B
¬(A ∨ B) = ¬A ∧ ¬B

半加算器の設計

要求仕様：

  • 入力：A, B（各1ビット）
  • 出力：S（和）、C（桁上がり）

論理式：

  • S = A ⊕ B（XOR）
  • C = A ∧ B（AND）
